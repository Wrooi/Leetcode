# Easy Problems

This directory contains solutions to LeetCode problems classified as **Easy** difficulty.

## Organization

Each problem should be organized in its own folder or file with the following naming convention:
- `<problem-number>-<problem-name>/` or `<problem-number>-<problem-name>.<extension>`

## Example Structure
```
Easy/
├── 1-two-sum/
│   ├── solution.py
│   └── README.md
├── 9-palindrome-number/
│   ├── solution.java
│   └── README.md
```

## Adding a New Solution

1. Create a new folder with the problem number and name
2. Add your solution file(s)
3. Optionally add a README.md with:
    - Problem description
    - Approach explanation
    - Time and space complexity
    - Link to the LeetCode problem
